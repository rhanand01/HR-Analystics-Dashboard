# 📊 Human Resources Dashboard | Tableau

An interactive Human Resources Analytics Dashboard built using **Tableau** to help HR teams monitor workforce demographics, employee distribution, hiring trends, salaries, education, performance, and detailed employee records.

The dashboard enables users to analyze employee information at both organizational and individual levels through interactive visualizations and filters.
